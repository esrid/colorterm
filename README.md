# ColorTerm 🎨

A simple tool to help you build and export terminal color schemes. It handles the math for color contrast so you don't have to, and it helps you get your theme into your favorite terminal or editor quickly.

**Try it out:** [https://colorterm.fly.dev](https://colorterm.fly.dev)

## What it can do:

-   **Randomize with Style:** Generate themes based on color theory (Analogous, Complementary, etc.).
-   **Image-to-Theme:** Upload a picture and it will pick out the colors for you.
-   **Contrast Fixing:** A one-click button to make sure your text is readable against your background (WCAG AA/AAA).
-   **Lots of Exports:** Supports Foot, Ghostty, Kitty, Alacritty, Helix, Neovim, VS Code, and more.
-   **Live Preview:** An interactive little terminal to see how the colors look with `ls`, `git`, etc.
-   **No Backend:** Everything stays in your browser. You can share your theme by just copying the URL.

## How to run it locally:

```bash
# Install
bun install

# Run
bun run dev
```
